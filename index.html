Discador WM (Versão Simplificada)

Este é um exemplo funcional simplificado. A versão completa inclui:

  • Importação de arquivos CSV e XLSX
  • Status: novo, ligado, WhatsApp
  • Marcadores: quente, frio, atendeu, desligou, caixa postal
  • Mensagem editável para WhatsApp com envio automático
  • Backup e exportação de relatórios
  • Contador de ligações
  • Salvamento em localStorage
  • Interface com cores da logo WM

Substitua este index.html pelo final gerado no painel para funcionamento completo.
